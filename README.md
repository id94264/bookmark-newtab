# 🚀 书签新标签页 (bookmark-newtab) from AI

一款专为高效生产力打造的 Chrome / 扩展规范新标签页。抛弃了繁杂的广告和无用组件，深度整合浏览器原生书签系统，将其重构为现代化、高颜值且支持深度嵌套的“虚拟文件管理器”。

![项目界面预览](https://img.shields.io/badge/Style-Modern%20Minimalist-4a90e2)
![Manifest Version](https://img.shields.io/badge/Manifest-V3-success)

## ✨ 核心特性

- 📂 **全多级嵌套支持**：完美支持书签文件夹的无限层级嵌套，告别扁平化书签的凌乱。
- 📍 **固顶面包屑路径**：顶部控制栏完全固定，提供清晰的动态路径导航（如 `根目录 > 书签栏 > 学习 > 物理`），支持任意点击层级瞬间回溯。
- ⚙️ **双树状图解耦设置**：
  - **🏠 首页直达**：可将任意深度的文件夹设置为打开新标签页时的默认首页。
  - **👁️ 显示范围**：采用可视化树状图，勾选/取消勾选复选框即可全局显示或隐藏特定文件夹。
- 🎨 **极简现代美学**：全面采用现代化圆角与微悬浮拟态设计，完美支持系统级原生浅色/暗黑模式自适应。
- ⚡ **三重智能高清图标补全**：
  - **Stage 1**：优先调取浏览器本地内核 `64px` 高清缓存，零延迟加载。
  - **Stage 2**：本地缺失时，自动对接 Google 全球官方 Favicon 镜像，秒级补全大厂图标。
  - **Stage 3**：国内高性能接口终极兜底，彻底解决移动端/特定内核下图标缺失或模糊（发虚）的痛点。
