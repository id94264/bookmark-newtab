# 书签新标签页浏览器插件

一个简洁美观的浏览器新标签页扩展，用于展示和管理您的收藏夹书签。

## ✨ 功能特性

- 📁 **层级展示**：以文件夹树状结构展示所有书签
- 🔍 **实时搜索**：支持按标题和URL搜索书签
- 🎨 **精美界面**：渐变背景 + 毛玻璃效果，现代化UI设计
- ⏰ **时间显示**：实时显示当前日期和时间
- 📊 **书签计数**：显示每个文件夹内的书签数量
- 🖱️ **交互友好**：点击文件夹可展开/折叠，悬停动画效果
- 📱 **响应式设计**：完美适配各种屏幕尺寸

## 📦 文件结构

```
bookmark-newtab/
```

## 🚀 安装方法

### Chrome/Edge 浏览器安装

1. 打开浏览器扩展管理页面：
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`

2. 开启右上角的「开发者模式」

3. 点击「加载已解压的扩展程序」

4. 选择本插件的 `bookmark-newtab` 文件夹

5. 安装完成！打开新标签页即可使用

## 🎯 使用说明

- **搜索书签**：在顶部搜索框输入关键词，实时过滤书签
- **展开/折叠**：点击文件夹标题即可展开或折叠内容
- **打开书签**：直接点击书签链接即可在新标签页打开
- **自动聚焦**：打开新标签页后搜索框自动聚焦，可直接输入搜索

## 🔧 技术栈

- **Manifest V3**：Chrome 扩展最新标准
- **Vanilla JavaScript**：原生JS，无框架依赖
- **CSS3**：渐变、毛玻璃、动画效果
- **Chrome Bookmarks API**：调用浏览器书签接口

## 📝 自定义修改

您可以根据个人喜好修改以下内容：

1. **主题颜色**：修改 `style.css` 中 `body` 的 `background` 渐变
2. **布局样式**：调整各元素的 padding、margin、border-radius 等
3. **功能扩展**：在 `script.js` 中添加更多功能
